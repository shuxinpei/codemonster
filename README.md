# codemonster
CodeMonster是一个帮助CS专业学生统计代码行数的工具<br>
原本是SSD4作业，课程要求只要堆出好看的界面就OK了，在好看的界面的基础上完成了所有的功能。<br>
通过仿搜狗输入法的界面来完成项目<br>
使用jnative以及jna完成底层输入监听，对回车进行数量统计，以及对一些特殊情况下的回车进行规避。<br>
同时含有大量的交互上的改进，包括数据可视化等等...<br>
